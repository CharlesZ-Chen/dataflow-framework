Description

Design

  • WorkList

    WorkList now designed as an inner abstract class of AbstractAnalysis, with an abstract method: abstract DFOComparator(). This could allow Forward Analysis has a min heap of DFS order of Block list, while Backward Analysis has a max heap of DFS order (which means reversed) of Block list.

Note

  • about ThenStore and ElseStore:

    I don't think BackWard needs to distinguish between then and else branch, and thus the Backward analysis should have a simpler store map, i.e. only stores<Node, Store>
